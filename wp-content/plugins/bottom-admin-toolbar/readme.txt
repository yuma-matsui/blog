== Description ==

This plugin will change the position of your admin toolbar to the bottom.
That's sometimes complicated to develop / design a header when you have that sticky top bar.
It's now easier to do whatever you want (Header, Footer, Animations..)

ðŸ’¡ You can hide that bar by pressing Shift + Arrow down.

== Installation ==

1. Upload the `bottom-admin-toolbar` folder to the `/wp-content/plugins/` directory
2. Activate the plugin through the 'Plugins' menu in WordPress
